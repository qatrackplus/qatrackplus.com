Why you should choose QATrack+

Developed for the needs of a modern radiation therapy clinic

QATrack+ has many features to make your life easier:

  • Replace paper forms & spreadsheets with a unified system
  • QC test types including numerical, boolean, calculated, file upload & analysis, and more
  • Multiple user groups (Physicists, Therapists, etc.) and group specific permissions via a configurable authorization system
  • Built in review & approval functionality for QC data
  • Fast and easy data trending
  • Scheduling system to keep on top of your QC program.
  • Did we mention it‘s free?.

  • Configurable

    QATrack+ is highly configurable allowing you to tailor your installation to the needs of your clinic. You define the test data that you want to record! QATrack+ is not tied to any specifc vendor or device and is currently being used for QA of Linacs, Tomotherapy, Cyberknife, CT Simulators and more.

  • Reliable

    QATrack+ was developed and implemented clinically at The Ottawa Hospital Cancer Centre, one of the largest cancer centres in Canada. QATrack+ has proven to be extremely reliable in this setting, recording hundreds of thousands of QC test results without breaking a sweat.

  • Completely Free

    QATrack+ is open source software that you are free to download and use in your clinic. There is no initial licensing costs and upgrades are free for ever. Community support is provided via an online forum. Development & and bug reports can be tracked via the online source code repository.

Getting Started

Technical Requirements

QATrack+ is a cross platform server application and requires either a Linux, Windows or Mac server and either a PostgreSQL, MySQL, or Microsoft SQL database (MSSQL).

QATrack+ can be accessed via Firefox, Chrome, IE8+, Safari and other modern browsers.

Installing & Deploy QATrack+

Detailed instructions for downloading, installing & deploying the software are available on the QATrack+ Wiki

Contributing to the QATrack+ project

QATrack+ welcomes contributions!

QATrack+ is a free and open source (MIT License) community supported software project and we are delighted to accept contributions from the Medical Physics community. Ways you can contribute to the QATrack+ project:

  • Write some code for some missing functionality and contribute it back to the project
  • Report bugs and request features with our issue tracker
  • Help document the project on our wiki
  • Participate and answer questions in our online support forum
